Machine Learning

  • Clustering Antarctic Penguin Species (DataCamp Project) - Arctic Penguin Exploration: Unraveling Clusters in the Icy Domain with K-means Clustering

Data Analysis

  • Exploring the Bitcoin Cryptocurrency Market (DataCamp Project) - Explore bitcoin and other cryptocurrency data. Clean the dataset by discarding cryptocurrencies without market capitalization, compare Bitcoin with other currencies, and prepare data for visualization.
  • Exploring 67 Years of LEGO(DataCamp Project) - This project explores a comprehensive dataset of every Lego block ever produced.
  • TV, Halftime Shows, and the Big Game (DataCamp Project) - This project explores how the key elements of the Super Bowl—game outcomes, TV viewership, ad costs, and halftime performances—interact using data from 52 Super Bowls through 2018, sourced from Wikipedia.
  • Real-time Insights from Social Media Data (DataCamp Project) - This project examines Twitter's influence by analyzing real-time trends and moods from worldwide and U.S. trending topics. Using pre-collected data from Twitter’s API, participants will focus on extracting insights rather than handling authentication, exploring how social media reflects global conversations.

Visualization

  • A Visual History of Nobel Prize Winners (DataCamp Project) - This project analyzes over a century of Nobel Prize data using Python, revealing patterns and potential biases through pandas-driven manipulation and Seaborn visualizations.
  • Recreating John Snow's Ghost Map (DataCamp Project) - This project recreates Dr. John Snow’s pioneering hand-mapped Cholera case study, which identified outbreak origins and laid the foundation for modern epidemiology.
